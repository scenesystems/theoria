Grow `effect-math` from a general math foundation into a much more practical toolkit for analysis, modeling, and scoring work.

This release adds new areas such as FFTs, ODE solving, root-finding, inferential reporting, and power analysis. It also adds reusable score-building helpers so other packages can turn raw measurements into consistent summaries and comparisons without each package inventing its own math layer.

Alongside the new capabilities, it strengthens the package’s release governance so the public API stays clear and dependable as more statistical and numerical tools move into the library.
